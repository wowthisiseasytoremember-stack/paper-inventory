# Future Features / Backlog

### 1. Analyze Group (Bundle Intelligence)
*   **Concept:** A feature that allows the user to select multiple scanned items (e.g., a stack of 1990s comics, or a collection of D&RGW correspondence) and hit an "Analyze Group" button.
*   **AI Goal:** A smart model evaluates the items *in relation to each other*, looking for connections that increase their collective value.
*   **Examples:**
    *   Identifying a continuous run of a specific comic character or storyline.
    *   Connecting a freight receipt with a corresponding internal memo from the same agent.
    *   Realizing that three separate maps form a complete survey of a specific county.
*   **Outcome:** Suggests "Bundle" lots and provides a combined valuation and "Lot Title".

### 2. Push to Inventory
*   **Concept:** A one-click bridge from the "Research Terminal" to the actual permanent Inventory/Listing app.
*   **Workflow:** After making a positive "Buy" decision on an item, the user clicks this button to transfer the item's image, structured metadata, and generated tags directly into their primary inventory management system.
